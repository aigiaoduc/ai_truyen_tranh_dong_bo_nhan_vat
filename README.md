# Trình Tạo Kịch Bản Truyện AI

Ứng dụng này sử dụng AI của Google Gemini để giúp các nhà văn, người kể chuyện và giáo viên biến những ý tưởng đơn giản thành kịch bản truyện tranh hoàn chỉnh. Từ một ý tưởng, ứng dụng sẽ tạo ra tiêu đề, cốt truyện, hồ sơ nhân vật nhất quán và các câu lệnh (prompt) tạo ảnh chi tiết cho từng cảnh.

## Tính năng chính

-   **Phát triển ý tưởng:** Biến một câu ý tưởng thành một câu chuyện có cấu trúc với số lượng cảnh tùy chỉnh.
-   **Phân tích kịch bản:** Phân tích kịch bản có sẵn của bạn để tạo ra tài sản sản xuất.
-   **Nhất quán nhân vật:** Tạo hồ sơ nhân vật chi tiết và đảm bảo ngoại hình của họ nhất quán trong tất cả các câu lệnh.
-   **Tạo lệnh chuyên sâu:** Sinh ra các câu lệnh tạo ảnh mạnh mẽ, bao gồm chỉ đạo nghệ thuật, góc máy và ánh sáng.
-   **Đa ngôn ngữ:** Cung cấp cả câu lệnh tiếng Việt và tiếng Anh.
-   **Tùy chỉnh phong cách:** Hỗ trợ nhiều phong cách minh họa khác nhau.

## Hướng dẫn sử dụng

Để sử dụng ứng dụng, bạn cần có một **Gemini API Key**.

### Bước 1: Lấy Gemini API Key

1.  Truy cập [Google AI Studio](https://aistudio.google.com/app/apikey).
2.  Đăng nhập bằng tài khoản Google của bạn.
3.  Nhấp vào nút "**Create API key**".
4.  Sao chép (copy) API key vừa được tạo. Đây là một chuỗi ký tự dài, ví dụ: `AIzaSy...`.

### Bước 2: Thêm API Key vào ứng dụng

1.  Mở ứng dụng [Trình Tạo Kịch Bản Truyện AI](https://main--character-consistent-comic-prompt-generator.googleplex.com/).
2.  Ở đầu giao diện bên trái, bạn sẽ thấy mục **"Quản lý API Key"**. Hãy nhấp vào đó để mở rộng.
3.  Dán API key bạn vừa sao chép vào ô nhập liệu.
4.  Nhấp vào nút **"Lưu API Key"**.

Sau khi lưu thành công, ứng dụng đã sẵn sàng để sử dụng. Key của bạn sẽ được lưu trữ trong trình duyệt (localStorage) cho những lần truy cập sau.

### Bước 3: Sáng tạo câu chuyện

1.  Chọn loại đầu vào là **"Ý Tưởng"** hoặc **"Câu Chuyện"**.
2.  Nhập ý tưởng hoặc kịch bản của bạn.
3.  Tùy chỉnh số lượng cảnh và phong cách minh họa.
4.  Nhấp vào nút **"Tạo Gợi Ý"**.
5.  Chờ trong giây lát để AI xử lý và xem kết quả!
